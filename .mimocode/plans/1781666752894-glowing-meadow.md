# 方形表盘第六行不偏移

## 问题
当前代码中，日历第六行（row===5）会向右偏移一个单元格宽度（CELL_W）。
用户希望方形表盘时不执行这个偏移。

## 代码位置
`page/gt/holiday/index.page.js` 第439行：
```javascript
const offsetX = (row === 5) ? CELL_W : 0;
```

## 修改方案

### 1. shared-styles.js
添加配置项：
```javascript
export const OFFSET_LAST_ROW = true;  // 默认圆形偏移
```

### 2. index.page.r.layout.js
导出配置（保持 true）

### 3. index.page.s.layout.js
覆盖配置为 false（方形不偏移）

### 4. index.page.js
导入配置并使用：
```javascript
const offsetX = (row === 5 && OFFSET_LAST_ROW) ? CELL_W : 0;
```

## 验证
- 方形表盘第六行不偏移
- 圆形表盘第六行保持偏移
